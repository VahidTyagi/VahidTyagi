## 🤖 AI-First Testing Philosophy

I don't just use AI as a shortcut — I integrate it as a **force multiplier** throughout the testing lifecycle:

- 🧠 Prompt-driven test design — using ChatGPT, Claude & Gemini for scenarios, edge cases, and negative flows early in the cycle  
- ⚙️ AI-assisted automation — GitHub Copilot & Playwright Codegen for faster test creation while keeping logic ownership intact  
- 🔍 LLM-driven risk analysis — identifying high-risk areas via AI-assisted exploratory testing  
- 🌐 Agentic QA exploration — actively learning MCP Server concepts and intelligent automation patterns
